# Modeling tracer in positron emission tomography

The python files in this repository were produced for a master's level mathematical modeling project course (11 ECTS). The project involved developing a new PET scanning compartment model based on an existing one, with the goal of modeling the activity of the tracer more accurately.

The project was completed in a team of 3 students, where I was one of them. The Python programs were also developed as a group effort.
